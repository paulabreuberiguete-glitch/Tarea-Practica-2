102 HTML Tags Reference

  • <a> - Defines a hyperlink. Example: <a href='https://example.com'>Link</a>
  • <abbr> - Defines an abbreviation or acronym. Example: <abbr title='Hypertext Markup Language'>HTML</abbr>
  • <address> - Defines contact information for an author or owner of a document. Example: <address>123 Main St</address>
  • <area> - Defines a clickable area inside an image map. Example: <area shape='rect' coords='34,44,270,350' href='{' target='_blank'>Click</area>
  • <article> - Defines an independent, self-contained piece of content. Example: <article><h2>Article Title</h2><p>Article content...</p></article>
  • <aside> - Defines content aside from the main content. Example: <aside>Related Links</aside>
  • <audio> - Defines sound content. Example: <audio controls><source src='audio.mp3' type='audio/mpeg'>Your browser does not support the audio tag.</audio>
  • <b> - Defines bold text. Example: <b>Bold text</b>
  • <base> - Specifies a base URL for relative URLs in a document. Example: <base href='https://example.com/'>
  • <bdi> - Isolates a part of text that might be formatted in a different direction. Example: <bdi>This text is not translated!</bdi>
  • <bdo> - Overrides the current text direction. Example: <bdo dir='rtl'>This text is right-to-left.</bdo>
  • <br> - Inserts a single line break. Example: <br>
  • <button> - Defines a clickable button. Example: <button>Click me</button>
  • <canvas> - Defines a drawing area. Example: <canvas id='myCanvas' width='200' height='100'></canvas>
  • <caption> - Defines a table caption. Example: <table><caption>Table Caption</caption>
  • <cite> - Defines a citation. Example: <cite>Shakeapeare</cite>
  • <code> - Displays a single line of code. Example: <code>var x = 5;</code>
  • <col> - Specifies column properties for an element. Example: <col style='background-color:lightgray'>
  • <colgroup> - Specifies a group of one or more columns in a table for formatting. Example: <colgroup><col style='background-color:gray'></colgroup>
  • <data> - Links a given content with a machine-readable translation. Example: <data value='12345'>12345</data>
  • <datalist> - Specifies a list of pre-defined options for input controls. Example: <input list='browsers'><datalist id='browsers'><option value='Chrome'><option value='Firefox'></datalist>
  • <dd> - Defines a description of a term in a description list. Example: <dd>Description</dd>
  • <del> - Defines text that has been deleted from a document. Example: <del>Deleted text</del>
  • <details> - Defines additional details that the user can view or hide. Example: <details><summary>More info</summary>Details...</details>
  • <dfn> - Represents the defining instance of a term. Example: <dfn>HTML</dfn>
  • <dialog> - Defines a dialog box or window. Example: <dialog open>This is a dialog</dialog>
  • <div> - Defines a section or a division in an HTML document. Example: <div><p>Some text</p></div>
  • <dl> - Defines a description list. Example: <dl><dt>Term</dt><dd>Definition</dd></dl>
  • <dt> - Defines a term in a description list. Example: <dt>Term</dt>
  • <em> - Defines emphasized text. Example: <em>This text is emphasized</em>
  • <embed> - Defines a container for an external application or interactive content (like a plug-in). Example: <embed src='file.swf'>
  • <fieldset> - Groups related elements in a form. Example: <fieldset><legend>Personal Info</legend><p>Name: </p></fieldset>
  • <figcaption> - Defines a caption for a
  • <figure> - Specifies self-contained content, like an image with a caption. Example: <figure><img src='image.jpg'><figcaption>Caption</figcaption></figure>
  • <footer> - Defines a footer for a document or section. Example: <footer><p>Footer content</p></footer>
  • <form> - Defines an HTML form for user input. Example: <form><input type='text'></form>
  • <h1> to <h6> - Defines HTML headings. Example: <h1>Heading 1</h1> ... <h6>Heading 6</h6>
  • <head> - Contains meta-information about the document. Example: <head><title>Document Title</title></head>
  • <header> - Defines a header for a document or section. Example: <header><h1>Header Content</h1></header>
  • <hr> - Defines a thematic change in the content. Example: <hr>
  • <html> - Defines the root of an HTML document. Example: <html><head></head><body></body></html>
  • <i> - Defines italic text. Example: <i>Italic text</i>
  • <iframe> - Defines an inline frame. Example: <iframe src='https://example.com'></iframe>
  • <img> - Embeds an image in a document. Example: <img src='image.jpg' alt='Description'>
  • <input> - Defines an input control. Example: <input type='text'>
  • <ins> - Defines text that has been inserted into a document. Example: <ins>Inserted text</ins>
  • <kbd> - Defines keyboard input. Example: <kbd>Ctrl + C</kbd>
  • < - Defines a label for an element. Example: <label for='name'>Name</label><input type='text' id='name'>
  • <legend> - Defines a caption for a
    element. Example: <fieldset><legend>Personal Info</legend></fieldset>
  • <li> - Defines a list item. Example: <li>List item</li>
  • <link> - Defines the relationship between a document and an external resource (most used for stylesheets). Example: <link rel='stylesheet' href='style.css'>
  • <main> - Defines the main content of a document. Example: <main><p>Main Content</p></main>
  • <map> - Defines an image map. Example: <map name='mapname'><area shape='rect' coords='34,44,270,350' href='#'></map>
  • <mark> - Represents text that has been highlighted or marked. Example: <mark>Marked text</mark>
  • <meta> - Defines metadata about an HTML document. Example: <meta charset='UTF-8'>
  • <meter> - Defines a scalar measurement within a range. Example: <meter value='0.6' min='0' max='1'>60%</meter>
  • <nav> - Defines navigation links. Example: <nav><a href='#'>Home</a></nav>
  • <noscript> - Defines an alternate content for users that have disabled scripts in their browser or have no script support. Example: <noscript>JavaScript is not enabled</noscript>
  • <object> - Defines an embedded object. Example: <object data='file.pdf' type='application/pdf'></object>
  • <ol> - Defines an ordered list. Example: <ol><li>First item</li></ol>
  • <optgroup> - Defines a group of related options in a drop-down list. Example: <select><optgroup label='Fruits'><option value='apple'>Apple</option></optgroup></select>
  • <option> - Defines an option in a drop-down list. Example: <option value='volvo'>Volvo</option>
  • <output> - Defines the result of a calculation. Example: <output>Result</output>
  • <p> - Defines a paragraph. Example: <p>This is a paragraph.</p>
  • <param> - Defines a parameter for an object. Example: <param name='autoplay' value='true'>
  • <picture> - Defines a container for multiple images. Example: <picture><source srcset='image.webp' type='image/webp'><img src='image.jpg' alt='Image description'></picture>
  • <pre> - Defines preformatted text. Example: <pre>Preformatted text</pre>
  • <progress> - Represents the progress of a task. Example: <progress value='70' max='100'></progress>
  • <q> - Defines inline quotations. Example: <q>Quote here</q>
  • <rp> - Defines what to show in browsers that do not support ruby annotations. Example: <rp>( )</rp>
  • <rt> - Defines a ruby annotation. Example: <rt>Ruby text</rt>
  • <s> - Defines text that is no longer correct. Example: <s>Incorrect text</s>
  • <sample> - Represents a sample output. Example: <sample>Sample text</sample>
  • <script> - Defines a client-side script. Example: <script>alert('Hello world');</script>
  • <section> - Defines a section in a document. Example: <section><h2>Section title</h2></section>
  • <select> - Defines a drop-down list. Example: <select><option value='volvo'>Volvo</option></select>
  • <small> - Defines smaller text. Example: <small>This text is small</small>
  • <source> - Specifies multiple media resources for elements like
  • <span> - Defines a section in a document. Example: <span>Some text</span>
  • <strong> - Defines important text. Example: <strong>Important text</strong>
  • <style> - Defines style information for a document. Example: <style>body { background-color: lightblue; }</style>
  • <sub> - Defines subscript text. Example: <sub>Subscript</sub>
  • <summary> - Defines a summary for a
    element. Example: <summary>Details</summary>
  • <sup> - Defines superscript text. Example: <sup>Superscript</sup>
  • <table> - Defines a table. Example: <table><tr><td>Cell</td></tr></table>
  • <tbody> - Groups the body content in a table. Example: <tbody><tr><td>Cell</td></tr></tbody>
  • <td> - Defines a cell in a table. Example: <td>Table Cell</td>
  • <textarea> - Defines a multi-line text input. Example: <textarea>Text area content</textarea>
  • <tfoot> - Groups the footer content in a table. Example: <tfoot><tr><td>Footer</td></tr></tfoot>
  • <th> - Defines a header cell in a table. Example: <th>Header</th>
  • <thead> - Groups the header content in a table. Example: <thead><tr><th>Header</th></tr></thead>
  • <time> - Represents a specific period in time. Example: <time datetime='2022-01-01'T00:00:00>New Year's Day</time>
  • <title> - Defines a title for the document. Example: <title>Document Title</title>
  • <tr> - Defines a row in a table. Example: <tr><td>Row</td></tr>
  • <track> - Provides text tracks for
  • <u> - Defines text that should be styled as underlined. Example: <u>Underlined text</u>
  • <ul> - Defines an unordered list. Example: <ul><li>List item</li></ul>
  • <var> - Defines a variable in a mathematical expression. Example: <var>x</var>
  • <video> - Defines a video or movie. Example: <video controls><source src='video.mp4'>Your browser does not support the video tag.</video>
  • <wbr> - Suggests where in a text it would be ok to add a line break. Example: <wbr>
