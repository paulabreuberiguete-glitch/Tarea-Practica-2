## Introduction
HTML (HyperText Markup Language) is the standard markup language for creating web pages. It helps structure content on the web.

## HTML Tags Overview
HTML tags are the building blocks of HTML. They are used to create elements on a web page. Here are some categories of HTML tags:
- Structural Tags
- Formatting Tags
- Link Tags
- List Tags
- Table Tags
- Media Tags

## Detailed Documentation of HTML Tags
### 1. `<a>` - Anchor Tag
- **Description**: Defines a hyperlink, which is used to link from one page to another.
- **Usage**: `<a href="https://www.example.com">Visit Example</a>`  

### 2. `<div>` - Division Tag
- **Description**: Defines a division or section in an HTML document.
- **Usage**: `<div class="container">Content here...</div>`  

### 3. `<span>` - Span Tag
- **Description**: Used to group inline elements in a document.
- **Usage**: `<span style="color: red;">Red Text</span>`
